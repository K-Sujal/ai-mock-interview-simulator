# AI Mock Interview Simulator

An AI-powered mock interview website that generates technical interview questions and allows users to practice answering them.

## Features

- Random interview question generator
- React frontend interface
- Node.js backend API
- Simulates real interview environment
- Helps students prepare for technical interviews

## Tech Stack

Frontend:
- React
- JavaScript
- HTML
- CSS

Backend:
- Node.js
- Express.js

## Project Structure

AI_MOCK_INTERVIEW
│
├── frontend
│   ├── src
│   ├── public
│   └── package.json
│
└── backend
    ├── routes
    ├── services
    └── server.js

## How to Run

### Backend

cd backend  
npm install  
node server.js  

### Frontend

cd frontend  
npm install  
npm start  

Frontend runs on:

http://localhost:3000

Backend runs on:

http://localhost:5000
